Matrix Product Documentation

Installation

    $ git clone https://github.com/JimmyTournemaine/MatrixProducts.git
    $ make
    $ make clean

Running

Tree executables has been compiled :

  • generator to create or randomly generate matrix two by two. They will use to calculate their products.
  • product_v1 to calculate the products and store them in a file.
  • product_fair ditto, but threads are forced to be fairly distributed on processors.

Two bash scripts are available to simply run the executables : run and compare. For both scripts, the stderr channel is redirected to a log file to not overload your device display.

    $ bash run nbMult file_out

This first script generate nbMult products, store the generated matrix in a script-managed temporary file. Then, it run product and save results in the file_out file.

    $ bash compare nbMult file_out

This second script run the generation like previously but execute the two versions of products calculations to compare both run times.
